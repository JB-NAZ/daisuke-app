# ダイスケ (大学スケジュール管理)

大学生活を効率化するためのモダンでミニマルなスケジュール管理Webアプリです。
AIを活用した開発プロセスの一環として作成されました。

## 機能 (Features)
- **カテゴリー管理**: 課題, 出席, バイト, 行事, その他
- **カレンダー表示**: 月ごとのイベント確認
- **タスク管理**: 課題の登録・完了・削除
- **メモ機能**: 自動保存されるメモ帳
- **ローカル保存**: データはブラウザに保存されます

## 技術スタック (Tech Stack)
- HTML5
- CSS3 (Vanilla, CSS Variables)
- JavaScript (Vanilla)
- Font: Noto Sans JP

## 実行方法 (How to Run)
`index.html`をブラウザで開くだけで使用できます。
